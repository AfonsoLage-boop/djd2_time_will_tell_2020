# Time Will Tell

-> [Game Design Document]

## Project Requirements

* Genre: __1st Person Adventure__
* Single-player
* 3D

__Must have:__

* Interaction with Objects, Environment and/or NPCs
* Inventory
* Menus
* DEV Cheats
* Audio
* Teaser Trailer
* External assets documented

__Can't depend on:__

* Accuracy
* Reflexes
* Time Constraints
* Game Over
